# 🐶🐱 Dog vs Cat Classifier

A simple Dog vs Cat image classifier using a pretrained ResNet-18 model.

This is a demo application for a fine-tuned ResNet-18 model for classifying images of dogs and cats.

## How to use

Simply upload an image or use your webcam to get a prediction.

## Model

The model is a ResNet-18 model fine-tuned on a private dataset of dog and cat images. The model is trained to classify images into two classes: "Dog" and "Cat".

## Training

The model was trained for 20 epochs with a learning rate of 0.001.
